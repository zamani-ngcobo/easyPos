# easyPos POS System

## Overview
The easyPos project is a modern Point of Sale (POS) interface designed for efficient transaction management. It provides a user-friendly interface for managing products, processing payments, and applying discounts.

## Project Structure
```
easyPos
├── src
│   ├── index.html          # Main HTML document for the application
│   ├── css
│   │   └── styles.css      # CSS styles for the application
│   └── js
│       ├── app.js          # Main JavaScript logic for the application
│       └── payments.js      # Functions related to payment processing
├── package.json            # Configuration file for npm
├── .gitignore              # Files and directories to ignore by Git
└── README.md               # Documentation for the project
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd easyPos
   ```

2. **Install Dependencies**
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the Application**
   Open `src/index.html` in your web browser to view the POS interface.

## Usage
- Select products from the menu to add them to the transaction.
- View the transaction summary on the right side of the interface.
- Choose a payment method and finalize the transaction.
